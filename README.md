# Life_U
# 🎓 Student Life Organizer

A smart all-in-one system designed to help university students manage their daily life efficiently — including finances, study planning, and personal goals.

---

## 🚀 Overview

Student Life Organizer is a comprehensive application that helps students:

- 💰 Track and manage their expenses
- 🏦 Save money and achieve financial goals
- 📚 Organize study schedules
- 🎯 Set and track weekly goals
- 📊 Analyze productivity and spending habits

This project aims to improve students' productivity, financial awareness, and overall life organization.

---

## 🧠 Key Features

### 💵 Financial Management
- Track daily expenses
- Monthly budget planning
- Savings goals
- Spending analysis (charts & reports)

### 📖 Study Planner
- Create study schedules
- Task management system
- Reminders & notifications
- Track study progress

### 🎯 Goal System
- Set weekly/monthly goals
- Progress tracking
- Achievement system (badges & rewards)

### 📊 Smart Insights (Advanced Feature)
- Analyze spending habits
- Suggest better budget plans
- Track productivity trends
